# XOR 
背景：常规 xor 逻辑电路大概需要 8~12 个 mos 管，作者构思了一种 4 mos + 2 diode 的结构提供参考

```
                          +------------------+
                          |                  |
                 +--------|-------------+    |
                 |        |             |    |
            +----|--------|--------+    |    |
            |    |        |        |    |    |
[IO.A]------+--*[P]       |       [P]*--+    |
            |    |        |        |    |    |
            |    +---[>]--+--[<]---+    |    +-------[IO.C]
            |    |                 |    |
            +--*[N]               [N]*--+
                 |                 |    |
                 +-----------------+    |
                 |                      |
                [-]                     |
[IO.B]----------------------------------+

[>] 
[<] 均为肖特基二极管，箭头指向导通方向
[P] PMOS （* 为控制端）
[N] NMOS 
[-] 为 GND

```
