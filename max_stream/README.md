# 最大流
这是一篇针对**最大流**的研究。在讨论一个网络的两个间接相连节点之间的最大传输带宽时，我们通常会把它转换成最大流问题。

## 简介
```
S +--- 10MB/s ---> 1 --- 6MB/s ---> 3
  |                |                |
  |              6MB/s              |
  |                |                |
  |                V                |
  |                + <--- 3MB/s ----+
  |                |                |
  |                |              8MB/s
  |                |                |
  |                V                V
  +--- 2MB/s ----> 2 --- 5MB/s ---> T
```
已知节点间的带宽，求 1 秒钟从 S 可以传多少数据到 T ？这个问题已经有标准解法了，本文将讨论一种有趣的解法。先把这个问题先放一下，我们来看下列两种特殊情况

```
情况一：
S +--- 5MB/s ---> T
  |               A
  |               |
  +--- 3MB/s -----+

对于情况一，我们可以很容易得出 1 秒钟最多传送 5M + 3M -> 8M 的数据
因为 S 有两条路可以到达 T，也就是可以并行传输，我们暂时把这种结构叫做并联传输结构

情况二：
S +--- 5MB/s ---> 1 --- 3MB/s ---> T

对于情况二，似乎不好一眼看出答案，我们先用原始方法来解
令 S 到 1 耗费 A 秒，1 到 T 耗费 B 秒
令 S 最多传输 C MB 数据到 T
由于从 S 到 1 和 1 到 T 流出的数据量总是相等的所以有以下等式：
A + B = 1 秒
5A = 3B = C

A           = 3B/5
3B/5 + B    = 1
8B/5        = 1
B           = 5/8
A           = 3/8
C           = 15/8 MB

我们不妨把传输带宽看作电导（值越大也就表示通路的传输速度越快），带宽的倒数就是电阻，时间就是电压，数据量就是电流。
那么对于情况一就变成求 1/5 与 1/3 的电阻并联后的等效电导
Rc = 1/(1/Ra + 1/Rb) = 1/(1/(1/5) + 1/(1/3)) = 1/(5 + 3)
由于电阻表示的是带宽的倒数，所以 5 + 3 表示的才是真实的等效带宽

对于情况二，代入串联电阻公式：
Rc = Ra + Rb = 1/5 + 1/3 = 8/15
所以等效的带宽为 15/8
```
