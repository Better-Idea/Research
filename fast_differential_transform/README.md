# 快速微分变换(FDT)
今天看了`快速傅里叶变换(FFT)`的小视频，虽然一知半解，但我也产生了一个新的 Idea，或许也能有效改善多项式之间的乘法运算

## 思路
```
多项式方程：
A(x) = sum(ai * x^i), i ∈ [0, n)

FFT 主要步骤：
- n + 1 个坐标点可以确定唯一一个 n 阶多项式，所以 FFT 先把多项式系数向量变换成坐标点集
- 多项式 A0 * A1 -> (xi = xj, yi * yj) 自变量 x 不变，应变量 y 乘积的形式
- 把新的坐标点集合 -> 多项式

FDT 方法：
f(x) = k1 * x    -> f'(1 + Δ) ≈ k1 * (1 + 1 * Δ) = k1 + 1 * k1 * Δ
f(x) = k2 * x^2  -> f'(1 + Δ) ≈ k2 * (1 + 2 * Δ) = k2 + 2 * k2 * Δ
f(x) = k3 * x^3  -> f'(1 + Δ) ≈ k3 * (1 + 3 * Δ) = k3 + 3 * k3 * Δ
...
f(x) = kn * x^n  -> f'(1 + Δ) ≈ kn * (1 + n * Δ) = kn + n * kn * Δ

我们得到 d = A(1 + Δ) - A(1) = sum(i * ki) * Δ, i ∈ [0, n)
也就是在精度允许的范围内，我们可以在 O(n) 时间复杂度的情况下
通过 A(1 + i Δ) = A(1) + i * d, i ∈ [0, n)
计算出 n 个点的坐标
```
